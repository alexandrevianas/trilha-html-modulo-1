Onde aprender?

Em qual área você gostaria de atuar? Pois são muitas, sabe qual algum conhecimento ou vivência?

Não tive respostar para suas perguntar, então ele me orientou a buscar pelo site Dio.me onde ele fez alguns cursos e adorou o metodologia de formação. Fiquei muito interessado e me inscrevi.

Como funciona o HTML

Voltar

HTML é a linguagem usada para estruturar o conteúdo de páginas web. Não é uma linguagem de programação, mas sim uma linguagem de marcação, usada para descrever os elementos de uma página.

Aqui está um resumo de como ele funciona:

  3. HTML sozinho faz não faz tudo
